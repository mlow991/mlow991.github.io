Sum of All Multiples Calculator

This calculator can determine the sum of all multiples of a given number (up to two) within the bounds of a specified range. Simply input the number(s) you wish to calculate the sum of all multiples, specify the range, then press the calculate button to receive the result.

Note: The specified range is non-inclusive.

How to Operate

  • Input value into the 'Number' field
  • [Optional] To add another value (2 max) press the plus button
  • Specify the range
  • Press the calculate button to view result

How it Works

The calculator accepts up to 2 numbers to calculate the sum of multiples. Plain JavaScript and jQuery are used to calculate the sum of all multiples within the specified range.

Exceptions such as non-numbers, invalid ranges, and empty input fields are taken into account. When an exception is caught, a warning message is displayed to the user. Once all input fields are valid, the calculator simply loops through the range and adds every multiple within the range to the grand total.

In the case of two numbers provided by the user: If a number within the range is a multiple of both numbers, then it only added once to the total.
